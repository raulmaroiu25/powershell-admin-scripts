# PowerShell Admin Scripts

A collection of basic PowerShell scripts used for common IT Support and SysAdmin tasks in an enterprise environment.  
These scripts demonstrate automation for Active Directory user management.

## Scripts Included
- **Reset-UserPassword.ps1** – Resets a user's password and forces password change at next logon.
- **Check-GroupMembership.ps1** – Lists all AD groups a user belongs to.
- **Create-NewUserTemplate.ps1** – Template script for creating a new AD user.
- **Disable-User.ps1** – Disables a user account and moves it to a specific OU.
